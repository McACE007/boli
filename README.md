# Boli

An auction and bidding platform.

## Repository Structure

```
boli/
├── backend/    ← Spring Boot microservices + Node.js notification service
└── frontend/   ← React.js SPA (Vite, Tailwind, Framer Motion)
```

- **Backend** — see [`backend/README.md`](backend/README.md) for full setup and run instructions.
- **Frontend** — see [`frontend/README.md`](frontend/README.md) for dev server and build instructions.
